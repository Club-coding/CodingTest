> **[점프와 순간 이동](https://programmers.co.kr/learn/courses/30/lessons/12980?language=java)**

```java
public class Solution {
    public int solution(int n) {
        int ans = 1;

        while(n>1){ 
            if(n%2==1) ans++; //2로 나눈 나머지가 1이면 텔레포트 거리가 1부족함
            n/=2;
        }
        
        return ans;
    }
}
```
