> **[이진 변환 반복하기](https://programmers.co.kr/learn/courses/30/lessons/70129)**

```java
class Solution {
    public int[] solution(String s) {
        int[] answer = new int[2];

        while(!s.equals("1")){
            int size = s.length();
            s = s.replace("0","");
            answer[1] += size-s.length();
            s = Integer.toBinaryString(s.length());
            answer[0]++;
        }

        return answer;
    }
}
```
