> **[최소직사각형](https://programmers.co.kr/learn/courses/30/lessons/86491)**

```java
class Solution {
    public int solution(int[][] sizes) {
        int w = 0, h = 0;

        for(int[] card: sizes){
            if(card[0]>card[1]){
                if(card[0]>w) w = card[0];
                if(card[1]>h) h = card[1];
            }else{
                if(card[0]>h) h = card[0];
                if(card[1]>w) w = card[1];
            }
        }
        
        return w*h;
    }
}
```
